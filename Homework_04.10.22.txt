 First level: Укажите след. команды:

1.для полной очистки индекса
  git reset


 2.для просмотра разницы между раб.каталогом и посл. коммитом
  ???????????????


 3.для создания новой ветки и одновременного переключения в на нее
  git checkout -b 'name'


 4.для сравнения веток между собой
  git diff 'name..name'


 5.для удаления локальной ветки
  git branch -d 'name'


 6.для слияния веток
  git merge 'name'


 7.для перезаписи посл/коммита.
  git commit --amend -m '----------'


 8.Коротко опишите причину возникновения конфликтов при слиянии и порядок действий для их решения

  Конфликт слияния может возникать, когда объединенные ветви изменяют одну и ту же строку файла 
  по-разному или когда одна ветвь изменяет файл, а другая ветвь удаляет ее.
  Решение конфликта решаем вручную заходим через любой редактор выбираем то что нам нужно 
  и только тогда делаем слияние.
