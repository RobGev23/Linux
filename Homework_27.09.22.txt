--Коротко опишите, какие задачи с помощью механизма ветвления решаются в GIT? Зачем выполняется их слияние?

-Ветки нужны для того, чтобы программисты могли вести совместную работу над проектом и не мешать друг другу при этом.
 При создании проекта, Git создает базовую ветку. Она называется master/main веткой. Она считается центральной веткой.
-Слияние используется в Git, чтобы собрать воедино разветвленную историю. Команда git merge выполняет слияние отдельных направлений разработки,
 созданных с помощью команды git branch , в единую ветку.


--Укажите команды для:

--переименования веток.
& git branch --move  'name  new name' 

--создания и переключения на новую ветку;
& git branch <название ветки>
& git checkout <название ветки>

--удаление ветки.
& git rm <название ветки>
& git branch -d <название ветки>
& git branch -D <название ветки>

--слияния веток.
-переключиться на ветку, в которую вы хотите включить изменения, и выполнить команду git merge:
& git merge <название ветки>
